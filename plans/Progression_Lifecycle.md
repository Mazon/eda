# Progression Lifecycle (65 Sessions)

## Design Goal
*   **Duration**: 65 Sessions (3-4 hours each).
*   **End State**: Full Talent Pyramid (10 Talents) + Master Skills (~90%).
*   **Philosophy**: Low bookkeeping, steady "General" growth, organic "Skill" growth.

## 1. General Growth (Talents & Stats)
We use a **Flat XP** system for buying Talents and Attributes.

*   **Income**: **5 XP per Session**.
    *   *Total Campaign Income*: **325 XP**.
*   **Costs**:
    *   **Talents**:
        *   Tier 1: **5 XP** (1 Session)
        *   Tier 2: **10 XP** (2 Sessions)
        *   Tier 3: **15 XP** (3 Sessions)
        *   Tier 4: **20 XP** (4 Sessions)
        *   *Full Pyramid Cost*: 4x5 + 3x10 + 2x15 + 1x20 = **100 XP**.
    *   **Attributes**:
        *   +1 to one Attribute: **10 XP** (2 Sessions).
*   **Lifecycle Pacing**:
    *   **Sessions 1-10 (50 XP)**: Buy 4 Tier 1 Talents + 3 Attribute points. (Core Build online).
    *   **Sessions 11-25 (75 XP)**: Buy 3 Tier 2 Talents + 4 Attribute points. (Power spike).
    *   **Sessions 26-40 (75 XP)**: Buy 2 Tier 3 Talents + 4 Attribute points. (Specialization).
    *   **Sessions 41-50 (50 XP)**: Buy 1 Tier 4 Talent + 3 Attribute points. (Capstone).
    *   **Sessions 51-65 (75 XP)**: Spend remaining XP on Skill Unlocks, Utility Talents, or Final Stat pushes.

## 2. Skill Growth (The Fumble System)
We use the **Fumble Trigger** for skill increases.

*   **Trigger**: Critical Fumble (Doubles > Skill).
*   **Reward**: **Immediate +1 to that Skill**. (Simple, no rolling).
*   **Math Check**:
    *   **Rolls per Session**: ~20 significant rolls.
    *   **Early Game (Skill 30)**: ~7% Fumble rate -> 1.4 increases/session.
    *   **Mid Game (Skill 60)**: ~4% Fumble rate -> 0.8 increases/session.
    *   **Late Game (Skill 80)**: ~2% Fumble rate -> 0.4 increases/session.
    *   **Total Growth**: Over 65 sessions, a primary skill will grow by ~50-60 points.
    *   *Result*: Start 30 -> End 85-90. **Perfect Curve.**

## 3. Summary
This system hits your 65-session target mathematically without requiring any complex tracking.
*   **Session End**: "Everyone add 5 XP."
*   **During Play**: "Oh, Fumble? Add +1 to that skill."
*   **Result**: A character fully matured by the campaign finale.
